download_bilibili_video0.0.5版本来啦！
如果你想正常使用，那么请进入项目的\ffmpeg-7.0.1-full_build\ffmpeg-7.0.1-full_build\bin，记录下这个文件夹的绝对路径，并且将其存入系统的环境变量。如果进入你的系统shell，输入命令：
ffmpeg -version
如果没有出现错误，并且可以正常输出版本，ffmpeg便已经配置完毕。

此外，你需要一个python运行环境，访问https://www.pyhon.org，点击download，选择python版本，运行安装包，切记要勾选将python添加进环境变量path，其他按需选择即可，进入系统shell，运行命令：
python
如果没有出现错误，python环境便配置完毕了。（此部分可以自己在网上需按照教程视频观看）

输入命令：
pip install -r requirements.txt
安装所有依赖库，若没有出现错误，依赖库便安装完成了

切换至项目根目录，输入命令
python -m main

开始使用把！激动的少年郎！
